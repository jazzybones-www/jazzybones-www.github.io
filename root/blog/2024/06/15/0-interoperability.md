# the jazzybones word: interoperability

anybody can host a website at any point with any server infrastructure which can
be visited by anybody with any browser. it doesn't matter if i use nginx or
Apache, it doesn't matter if you're on Linux, MacOS, Windows, or BSD, it doesn't
matter if you're using Firefox or Chrome; all of this software speaks the same
language, and they can all speak to each other. this property of software is
called "interoperability".

not everything is like this; services like Discord are only accessible through
the official clients provided by Discord Inc. and homebrew clients that were
created by reverse engineering these official clients. we would say that Discord
is a non-interoperable service.

interoperability has its roots in the earliest days of the industrial
revolution. in the late 18th century, Eli Whitney invented the idea of
interchangable parts, where products would be made of individual components
which could be mass-produced and replaced one at a time.

interoperability creates a better experience for everybody. in the case of the
web, users can visit any website, server owners don't have to worry about
supporting the little quirks of each browser, programmers can scrape websites to
create cool stuff like search engines.

many social media sites have some form of interoperability in the form of
[APIs](https://en.wikipedia.org/wiki/API). before Elon Musk bought it, Twitter
had a very robust API; if you understood the language of the API, then you could
send and receive tweets outside of the official Twitter app. this was used for
many things, including an earthquake warning system, analysis of public
sentiment on various events, and many silly bots. then, Musk made the Twitter
API prohibitively expensive for most of these use cases, which (among other
things) has largely contributed to Twitter's decline under his leadership.

despite the lashback that Twitter received for increasing the cost of their API,
Reddit shortly followed suit, which led to a terribly organized protest by the
unpaid moderators of many of the most popular subreddits. researchers were
unable to use Reddit's data to analyze public sentiment, alternative Reddit
frontends designed for people with disabilities such as blindness became
endangered, Reddit's CEO lied about getting blackmailed by one of the site's
largest API users, and many of the site's bots had to shut down.

i really don't have a larger message here, i just think that interoperability is
cool, and the fact that it's in danger is bad.
