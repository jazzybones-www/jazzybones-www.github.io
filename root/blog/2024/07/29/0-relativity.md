# the relativity of truth

there's this super famous paradox called [the liar
paradox](https://www.youtube.com/watch?v=in4u2i9v4vg&t=9). basically, we take
the following statement:

> this sentence is false.

is this sentence true or false? if "this sentence is false" is a true statement,
then this statement is false, which means that it's not true. that's a
contradiction, so we know that the statement isn't true. on the other hand, if
this statement is false, then it must be true through the inverse of the
original logic. therefore this sentence is neither true nor false.

we might try and solve this by adding some third category, so instead of having
true and false, we have true, false, and foobar. if a statement is neither true
nor false, then it's foobar. therefore, this sentence is actually foobar. we
might circumvent this by changing our original statement to

> this sentence is not true.

this is called the strengthened liar paradox. this sentence is definitely
neither true nor false by the same logic as the other sentence, so it must be
foobar. but if this sentence is foobar, then this sentence is not true (it's not
false either, it's some third thing). if the sentence isn't true, though, then
the sentence must be true, which is a contradiction, so this sentence isn't
foobar either.

i think i have a solution to this problem. let's start by looking at a new
sentence:

> this sentence is true.

is this sentence true or false? the answer is that it could be both. if it's
true, then it's true, and if it's false, then it's false.

statements aren't just true or false on their own, they have to reference
something else. if i tell you "_The C Programming Language_ was written by
Dennis Ritchie and Brian Kernighan", there is no way for you to figure out if
i'm telling the truth just by looking at the statement itself. there's some
other thing in the world, i.e. the book, _The C Programming Language_, that
determines the truthiness of this statement. truthiness always has to be
deferred to some other authority.

even math defers its truthiness. [it's possible to do math without any numbers
bigger than seven](https://en.wikipedia.org/wiki/Modular_arithmetic). the
existence of other numbers is something that we just defined into existence. the
existence of other numbers has certain implications that lead to the rest of
math, but in the end it all boils down to these few arbitrary decisions, called
[axioms](https://en.wikipedia.org/wiki/Axiom). when we say "1+1=2", we're really
saying "1+1=2", assuming that there are numbers, and that there is a number
called 0, and a number called 1, and 1+1 does not equal either 0 or 1. by saying
that 1+1=2, we're also defining this new thing called "2", which is just a
shorthand for 1+1.

in the end everything probably boils down to empiricism, or axioms, or some
hidden assumptions. i don't know if the problem is necessarily self reference,
since it's possible to develop some isolated mathematical systems with their own
rules that seem internally consistent, i think it's more the lack of an outside
truth.
