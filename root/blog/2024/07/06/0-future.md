# the end of my streak and the future of this website

i wrote at least one thing on this website for 22 days straight. yesterday was
the end of that streak.

```
   _______
 .'       ',
/           \
|    RIP    |
|           |
| jb streak |
|           |
|  22 days  |
|           |
|___________|
```

this had to happen eventually. there really isn't a decent excuse, i just
straight up did not write. i feel a bit bad about it, but i don't know why. i'm
pretty sure i'm the only person who actually knows about this website, i don't
think i have any consistent readers, it's not like i'm letting anybody down by
abandoning this website, but i somehow feel compelled to keep this site alive by
continuing to add to it.

at some point this site will become abandonware. i don't know when or how it
will happen, it will probably take place slowly over a very long time as a lose
motivation to continue this site. eventually i'll find that jazzybones has
become just another one of my many abandoned projects.

that reality scares me, and i feel like missing a day is the first step of that.
even if nobody else cares about this site, _i_ care about it, and that's what
really matters.

i guess there's nothing left to do but build up another streak.
