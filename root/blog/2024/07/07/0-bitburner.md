# dang bitburner is really fun

i've just spent the past few hours playing
[bitburner](https://bitburner-official.github.io/). bitburner is a game where
you (among other things) write javascript programs to hack various servers
automatically. there's a standard interface that programs can use, and
[documentation](https://bitburner.readthedocs.io/en/latest/index.html) for that
interface.

this game is really convincing. obviously it's not entirely accurate, there's
this "NUKE" command that just hacks into any server and a "hack" command that
just gives you free money from a nuked server. i can overlook that, this is a
video game, not a ctf.

what's crazy to me is that almost everything else is spot on. like, there's a
whole ass vim emulator, and it works really well.

i love this sort of shit, just a ridiculously technical game with a lot of
freedom to do whatever you want.
